# Easy Exam
Service for ordering educational works.

# Technologies:
* Node.Js
* Express JS
* Handlebars
* MongoDB
* Mongoose
* WebSocket

# Installing
after the clone, create the .env file in the root folder

`
DATABASE_STRING=**your mongodb URI with username and password**
`

then write

`
 npm install
 `

# Running
`
npm start
`
